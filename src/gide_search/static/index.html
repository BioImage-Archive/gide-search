Search Help

GIDE Search lets you find studies across IDR, SSBD, and BioImage Archive using simple or advanced search syntax.

Simple Search

Just type words to search across all fields including titles, descriptions, authors, organisms, and imaging methods.

fluorescence Find studies mentioning fluorescence
mouse brain Find studies about mouse brain (matches both words)
Abdullah Find studies by author name
zebrafish confocal Find zebrafish studies using confocal microscopy

Field-Specific Search

Target specific fields using field:value syntax.

Field Description Example
title Study title title:fluorescence
description Study description description:cancer
source Data source (IDR, SSBD, BIA) source:BIA
keywords Study keywords keywords:microscopy

Boolean Operators

Combine search terms with boolean logic.

Operator Description Example
AND Both terms required mouse AND brain
OR Either term mouse OR human
NOT Exclude term mouse NOT liver
(...) Group terms (mouse OR human) AND brain

Advanced Syntax

Syntax Description Example
"..." Exact phrase "confocal microscopy"
* Wildcard (any characters) fluor* matches fluorescence, fluorescent, etc.
? Single character wildcard m?use matches mouse, mause
~ Fuzzy search (typo-tolerant) flourescence~ finds fluorescence
^N Boost term importance cancer^2 tumor

Example Queries

title:cancer AND source:BIA Cancer studies from BioImage Archive
"Mus musculus" AND confocal Mouse studies using confocal microscopy
source:IDR AND fluor* IDR studies with fluorescence-related terms
(brain OR neuron) AND NOT liver Brain/neuron studies, excluding liver

Using Filters

In addition to search syntax, you can use the sidebar filters to narrow results by:

  • Source: IDR, SSBD, or BioImage Archive
  • Organism: Filter by species (e.g., Mus musculus, Homo sapiens)
  • Imaging Method: Filter by technique (e.g., confocal, fluorescence)
  • Release Year: Filter by when the study was published

Filters combine with your search query using AND logic.
